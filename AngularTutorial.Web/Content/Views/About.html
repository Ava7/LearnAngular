Technical details

If you are interested in how this site works, you'll be happy to know that all the code is freely available on GitHub and licensed under the GPL-3.

As a brief overview, the backend runs on ASP.NET MVC and all of the backend code is done in C#. The frontend is coded in Angular (of course!) with the CSS being handled by Bootstrap with the Yeti theme from Bootswatch and icons from Font Awesome. Oh, and those lovely code editors are from Mozilla's outstanding Ace project, hooked in with some help from AngularUI. At some point I'll write a more in-depth summary of how this site is built on my blog and post a link here.
